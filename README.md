# diy gb cartridge

## overview

Gameboy homebrew. 32KB flash memory Cartridge.

## files

gb_cart1.comp : Cartridge's PCB data for 'CADLUS X'.

gb_wr.comp : Flash memory writer's PCB data for 'CADLUS X'.

wr.prg : Writing software for 'Pi STARTER(smileboom)'.
