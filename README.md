# diygbcartridge

## overview

Gameboy homebrew.

## files

gb_cart1.comp : Cartridge's PCB data for 'CADLUS X'.

gb_wr.comp : Flash memory writer's PCB for 'CADLUS X'.

wr.prg : Writing software for'Pi STARTER'.
